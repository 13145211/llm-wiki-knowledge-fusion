# 精读脚本（PDF 批量自动精读 · 油猴脚本）

三个脚本对应三个 AI 平台，功能一致，任选你常用的平台安装即可。**免费，无需 API Key** —— 直接驱动平台网页版。

| 脚本文件 | 平台 | 版本 | 平台特色 |
|------|------|------|------|
| [`doubao-pdf-uploader-pro.user.js`](doubao-pdf-uploader-pro.user.js) | [豆包](https://www.doubao.com) | 4.4.0 | SSE 流拦截 + MutationObserver 加速完成判定、可用平台默认总结按钮 |
| [`qianwen-pdf-uploader-pro.user.js`](qianwen-pdf-uploader-pro.user.js) | [千问](https://www.qianwen.com) | 4.0.0 | `showOpenFilePicker` 劫持注入文件、网络拦截 + DOM 双通道取回答 |
| [`deepseek-pdf-uploader.user.js`](deepseek-pdf-uploader.user.js) | [DeepSeek](https://chat.deepseek.com) | 1.0.0 | 自适应 DOM 探测、深度思考模式开关、思考链内容排除 |

## 安装（3 步）

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展（Edge / Chrome）
2. 点开上表对应的 `.user.js` 文件 → 点 **Raw** 按钮 → Tampermonkey 自动弹出安装页 → 确认安装
3. 访问对应 AI 平台网页版并登录，页面右上角出现控制面板

## 使用

1. 面板中**选择 PDF 文件夹**（或直接拖拽 PDF / 文件夹到页面）
2. 按需调整上传间隔、时间窗口等参数
3. 点 **▶ 开始**，之后全自动运行：

```
上传 PDF → 自动输入精读 Prompt → 等待 AI 回答 → 质量校验 → 保存 .md
   ↑                                                        │
   └────────────── 间隔等待后处理下一篇 ←──────────────────┘
```

精读笔记保存到**浏览器默认下载目录**，文件名 = `原PDF名_文献标题.md`。

## 内置精读 Prompt：通用文献七段法

适用于任意学科。产出 7 段结构化笔记：

> **基本信息 / 研究背景与问题 / 方法·技术路线 / 核心结果 / 创新点 / 局限与展望 / 原文摘要**

- 强制忠实原文：不编造数字、图文冲突以图为准、推断必须标注"合理推断 / 作者未述"
- 图表定量读取协议：谱图 / 显微影像 / 性能曲线 / 统计图表 / 拟合回归 / 表格
- 5 项自检：数字反查、措辞合规、图表全覆盖、引号溯源、零占位符
- **多角色变体自动轮换**（博士后 / 资深研究员 / 博士生 / 期刊审稿人 / 行业研发），降低回答同质化
- 可在面板 Prompt 编辑框改成你所在领域的专用版本

## 主要功能

| 功能 | 说明 |
|------|------|
| 批量队列 | 文件夹选择（File System Access API）/ 拖拽添加，自动记忆 |
| 断点续传 | 上传记录持久化，关页面、重启浏览器后接着跑 |
| 回答校验 | 过短 / 拒答自动判无效 → 冷却期 → 日常问题唤醒 → 继续 |
| 时间窗口 | 只在设定时段运行（如 08:00–22:00），窗口外自动暂停 |
| 精细控制 | 按序号上传（`1,3,5-10`）、点选上传、失败一键重试 |
| 持久化日志 | 导出进度报告 / 读取日志恢复进度 |

## 配置说明

所有默认参数集中在**每个脚本开头**的 `用户可调配置` 区块，逐项中文注释（上传间隔、最短等待、稳定判定、冷却时间、唤醒问题等）；运行时也可在面板中随时调整，自动保存。

## ⚠️ 注意

- 仅供个人学习与文献管理使用，请遵守各平台服务条款
- 高频操作可能触发平台风控，建议上传间隔 ≥ 10 分钟并设置时间窗口
- 脚本不收集、不上传任何数据到第三方
